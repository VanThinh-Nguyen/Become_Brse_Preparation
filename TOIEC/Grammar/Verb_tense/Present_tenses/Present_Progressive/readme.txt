Present Progressive: thì hiện tại tiếp diễn
------------------------------------------
------------------------------------------

Usage: 
    ☑ Hành động đang diễn ra ngay bây giờ:
        I am talking to you right now.
    ☑ Hành động đang diễn ra trong giai đoạn hiện tại:
        She is learning Japanese these days.
    ☑ Kế hoạch tương lai gần (đã lên lịch):
        We are meeting the manager tomorrow.
    ☑ Hành động gây khó chịu (dùng với "always")
       He is always coming late!

Cách chia: 
        ☑ Khẳng định:
            I                   + am  + V-ing
            He / She/ It        + is  + V-ing
            You / We / They     + are + V-ing

        ☑ Phủ định
            I                   + am  + not + V-ing
            He / She/ It        + is  + not + V-ing
            You / We / They     + are + not + V-ing
        
        ☑ Câu hỏi
            Am/Is/Are + S + V-ing?
    
    ※Cách thêm ing
        ☑ Thông thường : thêm ing  
            vd: play → playing

        ☑ Kết thúc bằng e : Bỏ e, thêm -ing
            vd: make → making

        ☑ Kết thúc bằng 1 nguyên âm + 1 phụ âm: Gấp đôi phụ âm, rồi thêm -ing
            vd: run → running

        ☑ Kết thúc bằng ie : Đổi ie thành y + ing
            vd: die → dying


Dấu hiệu nhận biết:
    ☑now / right now
    ☑at the moment
    ☑this week/month/year
    ☑look! / listen!
